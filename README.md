# **CineStitch 电影拼图工具**

**CineStitch** 是一个基于 React 和 HTML Canvas 开发的在线工具，专为电影爱好者和内容创作者设计。它允许用户将多张图片（如电影截图或剧照）拼接成一张具有电影画幅比例（如宽屏 CinemaScope 2.35:1）的长图，并提供灵活的裁剪和布局控制。

## **✨ 主要功能**

* **多图拼接：** 上传多张图片，按照上传顺序垂直拼接。  
* **电影画幅裁切：** 支持一键选择主流电影画幅比例（如 2.35:1, 2.39:1, 16:9）对每张图片进行统一裁切。  
* **垂直构图调整：** 通过滑块精确控制每张图片在裁切区域内的垂直位置，实现精确构图。  
* **布局参数控制：** 自由调整图片间的间距（Gap）、上下边距（Padding Y）和左右边距（Padding X），以及背景颜色。  
* **拖放上传：** 支持将图片文件直接拖放到指定区域快速上传。  
* **图层管理：** 方便地调整图片顺序、删除图片或清空全部图层。  

## **🎬 使用指南**

1. **上传图片：**  
   * 点击 **"点击或拖拽上传图片"** 区域，或直接将图片文件拖入该区域。  
2. **调整画幅比例：**  
   * 在 **全局设置** 面板中，选择您希望的电影画幅比例（默认为 2.35:1）。  
3. **调整图片顺序和裁剪：**  
   * 在 **图层顺序** 列表中，您可以：  
     * 使用 **上箭头** / **下箭头** 按钮调整图片在拼图中的垂直顺序。  
     * 使用 **垂直构图位置** 滑块，精细调整当前图片被裁切的垂直中心点，以突出您想要的画面内容。  
     * 使用 **垃圾桶图标** 删除不需要的图片。  
4. **调整布局：**  
   * 在 **全局设置** 面板中，调整 **间距 (Gap)**、**上下边距 (Padding Y)** 和 **左右边距 (Padding X)**，以定制拼图的视觉效果。  
5. **预览与导出：**  
   * 所有更改都会实时反映在右侧的 **预览区域**。  
   * 满意后，点击页面顶部的 **"导出图像"** 按钮，将最终的 JPG 拼图保存到本地。

## **⚙️ 技术栈**

* **前端框架：** React (通过 Babel 在浏览器中运行)  
* **样式框架：** Tailwind CSS  
* **核心绘图：** HTML Canvas API (用于图像处理和最终拼接)

## **⚠️ 注意事项**

* 为获得最佳导出效果，建议上传原始分辨率较高的图片。  
* 导出的最终图像宽度固定为 **2500px**。
